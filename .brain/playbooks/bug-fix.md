# Bug Fix Playbook

## Checklist

- [ ] **Reproduce**: Confirm the bug exists and understand symptoms
- [ ] **Analyze**: Find root cause (not just symptoms)
- [ ] **Plan**: Identify minimal fix (avoid scope creep)
- [ ] **Fix**: Apply surgical change
- [ ] **Test**: Verify fix resolves issue
- [ ] **Regression**: Ensure no new issues introduced
- [ ] **Document**: Add to known issues if relevant pattern
