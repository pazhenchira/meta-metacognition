# New Feature Playbook

## Checklist

- [ ] **Understand**: Read app_intent.md, essence.md, relevant context
- [ ] **Analyze**: Dispatch to Analyst if requirements unclear
- [ ] **Design**: Dispatch to Architect if complex/structural
- [ ] **Plan**: Break into LEGO components (single responsibility, clear interfaces)
- [ ] **Implement**: For each component:
  - [ ] Create/modify files
  - [ ] Test component in isolation
- [ ] **Review**: Dispatch to Reviewer for validation
- [ ] **Test**: Verify end-to-end functionality
- [ ] **Document**: Update docs if user-facing
- [ ] **Complete**: Update WORK_ITEM_TRACKER.md if applicable
