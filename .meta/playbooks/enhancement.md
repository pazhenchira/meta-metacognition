# Enhancement Playbook

## Checklist

- [ ] **Understand**: What improvement is needed and why?
- [ ] **Scope**: Define boundaries (what's in, what's out)
- [ ] **Impact**: Identify affected components
- [ ] **Plan**: Break into steps
- [ ] **Implement**: Make changes incrementally
- [ ] **Test**: Verify improvement achieved
- [ ] **Review**: Dispatch to Reviewer
- [ ] **Document**: Update relevant docs
