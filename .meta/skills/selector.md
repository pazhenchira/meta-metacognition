# Skill Selector

> Route work stages to the appropriate quality protocol.

## Process Skills (Always Check)

These skills fire based on the STAGE of work, not the topic.

### Investigation Framing
Use `investigation-framing.skill.md` **BEFORE** dispatching any investigation, deep analysis, root cause analysis, or comparison.

**Trigger**: Any request containing investigation, analysis, deep-dive, root cause, comparison, or "why is X different from Y?" language.

### Pre-Ship Review
Use `pre-ship-review.skill.md` **BEFORE** presenting any completed work to the user — completed analysis, authored document, recommendation, or work item completion.

**Trigger**: Any work item completion, deliverable presentation, or "ready for review" declaration.

### Structured Challenge
Use `structured-challenge.skill.md` **WHEN** evaluating high-stakes proposals, making irreversible recommendations, or stress-testing plans the user will act on.

**Trigger**: High-stakes proposals, irreversible decisions, or anything the user will present or approve.
